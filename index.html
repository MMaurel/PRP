PRP

Personal Ressources Planning

View project on GitHub

PRP -- is a fully responsive admin template based on AdminLTE.

PRP has been carefully coded with clear comments in all of its JS, LESS and HTML files. LESS has been used to increase code customizability.

Special Features:

  • Fully responsive
  • Enhanced for printing
  • Sortable dashboard widgets
  • 18 plugins and 3 custom plugins
  • Light weight and fast
  • Compatible with most major browsers
  • Full support for Glyphicons, Fontawesome and Ion icons
